# 附录 B 数学模型规范（Mathematical Model Specification）

Version 2.0

---

# B.1 设计目标

本规范用于统一所有美学评价指标（Metric）的数学计算方式。

原则：

1. 所有评分均输出 0~100 分。
2. 所有评分均为连续函数。
3. 禁止使用固定阈值直接扣分。
4. 相同类型指标必须采用相同数学模型。
5. 所有参数均可配置。

---

# B.2 数学模型分类

系统统一采用六类数学模型。

| 模型 | 适用场景 | 示例 |
|------|---------|------|
| Gaussian（高斯） | 存在最佳值 | 留白、颜色数量、字号比例 |
| Linear（线性） | 越小越好/越大越好 | 元素重叠、溢出 |
| Sigmoid（S型） | 达到阈值后收益递减 | 对比度、清晰度 |
| Spearman | 排名一致性 | 视觉焦点 |
| Kendall Tau | 顺序一致性 | 阅读顺序 |
| 欧氏距离 | 位置偏移 | 视觉重心 |

---

# B.3 Gaussian 模型

适用于：

存在最佳值的指标。

例如：

- 留白比例
- 图文比例
- 字号比例
- 色彩数量
- 图标面积

数学公式：

Score = 100 × exp(-(x-μ)^2 / (2σ²))

其中：

x：

当前值。

μ：

最佳值。

σ：

允许偏差。

特点：

偏离最佳值越远。

得分下降越快。

---

# B.4 Linear 模型

适用于：

越少越好。

或者：

越多越好。

例如：

元素重叠面积。

数学公式：

Score = max(0,100-kx)

其中：

k：

配置参数。

特点：

计算速度快。

解释性强。

---

# B.5 Sigmoid 模型

适用于：

达到一定水平以后。

继续提升意义不大。

例如：

文字对比度。

数学公式：

Score = 100/(1+exp(-k(x-x0)))

其中：

x0：

目标值。

k：

增长速度。

特点：

符合人眼感知。

---

# B.6 Spearman 排名相关系数

适用于：

视觉重点。

数学公式：

ρ = 1 - 6Σdi² / n(n²-1)

输出：

[-1,1]

转换：

Score = (ρ+1)/2 ×100

---

# B.7 Kendall Tau

适用于：

阅读顺序。

数学公式：

τ=(C-D)/(n(n-1)/2)

输出：

[-1,1]

转换：

Score=(τ+1)/2×100

---

# B.8 欧氏距离

适用于：

位置偏移。

例如：

视觉重心。

数学公式：

Distance=√((x-x0)²+(y-y0)²)

再经过 Gaussian 转换。

得到最终分数。

---

# B.9 Metric 与数学模型映射

| Metric | 数学模型 |
|----------|----------|
| 留白比例 | Gaussian |
| 边距均衡 | Gaussian |
| 元素间距 | Gaussian |
| 图文比例 | Gaussian |
| 图标比例 | Gaussian |
| 色彩数量 | Gaussian |
| 对比度 | Sigmoid |
| 清晰度 | Sigmoid |
| 重叠面积 | Linear |
| 元素溢出 | Linear |
| 视觉重心 | 欧氏距离+Gaussian |
| 焦点排序 | Spearman |
| 阅读顺序 | Kendall Tau |

统一使用上述模型。

禁止自行修改。

# B.10 参数配置

所有模型参数来自：

config/

例如：

metric.yaml

```yaml
Whitespace:

  mean:0.42

  sigma:0.08

Contrast:

  x0:4.5

  k:2.0

Overlap:

  k:120
```

代码禁止出现：

```python
score=80
```

所有参数必须来自配置。

---

# B.11 数值稳定性

所有公式应保证：

输出范围：

0~100。

异常值：

自动裁剪。

例如：

```python
score=max(0,min(100,score))
```

禁止：

NaN。

Inf。

负数。

---

# B.12 Explain 输出

每一个数学模型必须输出计算过程。

例如：

```text
Metric：

Whitespace

Current：

0.48

Ideal：

0.42

Deviation：

0.06

Model：

Gaussian

Score：

92.6
```

确保所有得分均可解释。
