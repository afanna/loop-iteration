# V3 第二次迭代规则优化说明

## 目标

本轮目标是让规则评测更贴近视觉老师评测，只优化可以用数学公式计算的维度；模板、视觉冲击、原创性不进入规则优化。

## 结果

| 阶段 | MAE | 规则-视觉偏差 |
| --- | ---: | ---: |
| 第二轮优化前 | 10.52 | -6.09 |
| 第二轮优化后 | 2.76 | +0.67 |

优化后绝大多数样本已经贴近视觉老师。最大残差仍是 `earbuds`：规则 52.09，视觉老师 42.19，误差 +9.90。

## 本轮改动

1. 布局占用率从只看 OCR/CV 元素框，改为 `max(元素框占比, 像素前景占比)`。
   - 背景色使用图片边缘像素估计。
   - 前景使用 LAB 色差阈值计算。
   - 解决 `care`、`sleep`、`status`、`marathon` 这类大色块/图形区域漏检导致的布局低估。

2. `whitespace` 和 `density` 从单点高斯改成区间容忍。
   - 留白可接受区间：0.10 到 0.65。
   - 密度可接受区间：0.35 到 0.90。
   - 这样不会因为卡片视觉丰富但元素框漏检而被误罚。

3. 总分维度权重调整。
   - `information`: 0
   - `layout`: 46
   - `visual`: 35
   - `consistency`: 8

4. 视觉维度内部权重调整。
   - 提高 `color_harmony` 和 `contrast`。
   - 降低 `visual_balance`。
   - 暂停 `visual_focus` 和 `hierarchy` 计权。

## 判断

第二轮核心误差来自两个方向：

- 好图低估：元素检测漏掉大面积图形/背景，导致布局密度和留白误判。
- 坏图高估：信息完整、对齐、重心、层级等数学指标会奖励“形式整齐但视觉品质弱”的图片。

本轮改动后，好图低估基本修正；坏图 `earbuds` 仍偏高，但已经从 +19.05 降到 +9.90。下一轮如果继续优化，建议只针对可公式化的“视觉材料不足/文本堆叠/低前景视觉承载”设计新指标，不碰模板和视觉冲击判断。
